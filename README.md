assignment 5 – Fun with JavaScript (CSC193A)
  This repository contains my solution for Assignment 5: Fun with JavaScript.
  The project includes an interactive web page where users can enter text and apply styling using JavaScript.
Files Included:
  fancifymytext.html – Main HTML page with UI controls
  fancifymytext.js – JavaScript file that applies text styling and effects
Features Implemented:
  Bigger! button increases text size
  FancyShmancy & BoringBetty radio buttons toggle styles
  Moo button uppercases text and adds “-Moo” to each sentence
